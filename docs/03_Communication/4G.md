# 4G

## 1. 基本概念

4G（Fourth Generation Mobile Network，第四代移动通信网络）在智能硬件中通常指通过蜂窝模组接入运营商网络。LTE（Long Term Evolution，长期演进技术）是 4G 设备中常见的无线接入技术名称。

## 2. 工作原理

设备通过 SIM（Subscriber Identity Module，用户身份识别模块）或 eSIM（embedded SIM，嵌入式用户身份识别模块）完成运营商鉴权，再通过蜂窝基站接入运营商核心网并获得数据连接。

## 3. 核心参数

- 支持频段：必须匹配目标国家、地区和运营商。
- LTE Category：影响理论速率和模组能力。
- 天线方案：影响信号质量和吞吐。
- SIM/eSIM 支持：影响供应链和激活流程。
- 功耗和待机策略：影响续航和热设计。

## 4. 参数影响

频段不匹配会导致无法入网或覆盖差。LTE Category 影响峰值能力，但实际体验还受运营商网络影响。天线设计不良会造成弱信号、掉线和高功耗。

## 5. 优点

- 覆盖范围广，适合移动联网。
- 不依赖用户路由器。
- 适合车辆、户外设备和远程设备。

## 6. 局限性

- 需要运营商资费、认证和激活流程。
- 弱信号场景功耗和发热会上升。
- 全球销售需要处理频段和认证差异。

## 7. 产品经理关注点

4G vs WiFi 的关键差异是联网依赖、资费、覆盖和功耗。产品经理必须明确目标市场、运营商、资费模型、激活方式、弱网体验和离线能力。

## 8. 常见应用场景

- 车载设备
- 户外摄像头
- 共享设备
- 工业网关

## 9. 需求文档需要明确内容

- 目标国家和运营商
- SIM/eSIM 策略
- APN 配置
- 网络注册状态展示
- 流量统计和限制
- 离线缓存和重传策略

## 10. 异常状态设计

- 无 SIM 或 SIM 错误
- APN 配置错误
- 网络注册失败
- 欠费或流量耗尽
- 弱信号掉线
- 漫游不可用
