# Magnetometer

## 1. 基本概念

Magnetometer（磁力计，用于测量磁场强度和方向的传感器）可用于电子罗盘、姿态融合和磁场异常检测。

## 2. 工作原理

磁力计测量三轴磁场分量，软件根据传感器坐标系、校准参数和姿态补偿计算磁场方向。Hard Iron Interference（硬磁干扰，由固定磁性材料造成的偏移）和 Soft Iron Interference（软磁干扰，由金属结构改变磁场分布造成的畸变）会影响测量结果。

## 3. 核心参数

- 测量范围：影响可测磁场强度上限。
- 分辨率：影响微小磁场变化识别能力。
- 噪声：影响输出稳定性。
- 输出数据率：影响刷新速度和功耗。
- 校准能力：影响实际可用性。

## 4. 参数影响

测量范围不足会在强磁环境饱和。噪声高会导致方向抖动。输出数据率过高会增加功耗，过低会降低响应速度。

## 5. 优点

- 可提供绝对磁场参考。
- 适合方向和磁异常功能。
- 可与加速度计、陀螺仪组成姿态融合方案。

## 6. 局限性

- 对结构材料和电流路径敏感。
- 需要校准和场景验证。
- 单独使用不能完整表达设备姿态。

## 7. 产品经理关注点

需要推动硬件、结构和算法共同评估磁干扰。Magnetometer vs Compass 的区别是传感器原始测量与产品功能输出的区别。

## 8. 常见应用场景

- 电子罗盘
- 姿态融合
- 磁开关检测
- 环境磁异常识别

## 9. 需求文档需要明确内容

- 是否输出原始磁场数据
- 校准流程
- 干扰判定阈值来源
- 坐标系定义
- 数据刷新率
- 异常提示策略

## 10. 异常状态设计

- 数据饱和
- 校准失败
- 磁干扰
- 传感器无响应
- 坐标轴方向错误
- 输出抖动超出产品可接受范围
