# Gyroscope

## 1. 基本概念

Gyroscope（陀螺仪，用于测量设备绕轴旋转角速度的传感器）常用于姿态检测、运动跟踪、防抖和导航辅助。

Angular Velocity（角速度，单位时间内角度变化速度）是陀螺仪的核心输出。

## 2. 工作原理

MEMS Gyroscope（Micro-Electro-Mechanical Systems Gyroscope，微机电系统陀螺仪）通过微结构振动和科里奥利效应感知旋转。软件对角速度积分可估算角度变化。

## 3. 核心参数

核心参数包括量程、噪声密度、零偏、零偏稳定性、采样率、带宽、功耗、温漂、封装方向和抗震能力。

Bias（零偏，在无旋转时传感器输出的偏移）会导致积分角度漂移。

## 4. 参数影响

零偏越大，长时间姿态估计漂移越明显。采样率影响快速动作捕捉。量程不足会在快速旋转时饱和。温漂会导致不同温度下输出变化。

Gyroscope vs G-Sensor 的差异是陀螺仪测旋转，加速度计测线性加速度和重力方向。

## 5. 优点

陀螺仪对短时间旋转变化响应快，适合动态姿态和防抖。

## 6. 局限性

陀螺仪积分会漂移，长期绝对方向需要与加速度计、磁力计或 GNSS 融合。

## 7. 产品经理关注点

需要明确动态场景、响应时间、漂移容忍度和是否需要传感器融合。不要把陀螺仪当成绝对方向来源。

## 8. 常见应用场景

常见于手机姿态、无人机、运动相机防抖、VR 控制器、机器人、车载姿态估计和运动识别。

## 9. 需求文档需要明确内容

PRD 需要明确量程、采样率、姿态输出、校准策略、温度范围、融合算法、异常提示和产测方法。

## 10. 异常状态设计

异常状态包括零偏过大、数据饱和、漂移、温度异常、传感器无响应、校准失败和坐标轴配置错误。
