# 硬件产品经理知识库

## 项目定位

硬件产品经理知识库用于沉淀硬件产品经理在产品定义、方案评审、需求文档、异常设计和项目复盘中需要反复查阅的技术知识。

知识库按硬件、存储、通信、传感器、操作系统、功能设计、PRD 和项目案例分类维护。

## 使用方式

使用顶部搜索框检索专业名词、接口、功能和异常状态。

使用左侧目录按分类浏览知识条目。

每篇知识文档按统一结构维护：基本概念、工作原理、核心参数、参数影响、优点、局限性、产品经理关注点、常见应用场景、需求文档需要明确内容和异常状态设计。
